# ELT3281: Microprocessor and Embedded System
## Description
A system can be developed for smart agriculture. A simple tasks of system is read temperature and humidity and show on LCD display. And show how to do after.

## Hardware
1. PIC16F877A
2. DHT11 sensor
3. LCD LM016L
